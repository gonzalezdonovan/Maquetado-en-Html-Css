# Proyecto Currículum Vitae (HTML y CSS)

## Descripción del proyecto
Este proyecto consiste en la creación de un sitio web que representa un **Currículum Vitae**, desarrollado utilizando únicamente **HTML** y **CSS**.  
El objetivo principal es mostrar información personal, académica y profesional de manera clara y ordenada.

El proyecto simula un currículum digital que podría ser utilizado con fines académicos o profesionales.

---

## Tecnologías utilizadas
- HTML5
- CSS3

---

## Estructura del proyecto
El proyecto está organizado de la siguiente manera:

- `index.html` → Contiene la estructura y el contenido del currículum.
- `styles.css` → Contiene los estilos visuales del sitio web.
- `ProfileImage.jpeg` → Fotografía de perfil utilizada en el currículum.
- `README.md` → Contiene informacion adicional del proyecto.

---

## Cómo ejecutar el proyecto

1. Descargar o clonar el repositorio.
2. Abrir el archivo `Index.html` en un navegador web.
3. Comenzar a agregar y administrar tareas.

No se requiere instalación adicional.

---

## Contenido del currículum
El currículum incluye las siguientes secciones:

- Fotografía de perfil
- Datos de contacto
- Formación académica
- Habilidades con porcentajes de dominio
- Experiencia laboral
- Experiencia en proyectos
- Intereses profesionales
- Software conocido con porcentaje de dominio
- Idiomas con porcentaje de dominio
- Dos referencias personales

---

## Características del diseño
- Diseño limpio y sencillo
- Uso de barras de progreso para representar niveles de habilidad
- Secciones organizadas para facilitar la lectura

---

## Objetivo del proyecto
El objetivo de este proyecto es poner en práctica:

- La estructura básica de HTML
- El uso de estilos con CSS
- La separación entre contenido y diseño
- La creación de un documento web con apariencia profesional
